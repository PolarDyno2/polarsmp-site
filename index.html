QUICK ANSWERS

Before you join.

Is PolarSMP cracked?

No. Java players need a valid Minecraft account. Bedrock players join through Floodgate with their Microsoft account.

Why can’t I join immediately?

The server is whitelisted. Ask a PolarSMP staff member to add your username before connecting.

Do Java and Bedrock players share the same world?

Yes. Progress, the economy, shops and the world are shared across both editions.

Which version should I use?

The server runs Paper 1.21.11. ViaVersion also helps nearby Java versions connect.
