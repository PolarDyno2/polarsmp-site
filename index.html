FAQ

The useful answers.

Do I need to apply?

No. PolarSMP is open access and the whitelist is disabled.

Is cracked Java supported?

No. Java players need a valid Minecraft account. Bedrock joins through Floodgate.

Do both editions share progress?

Yes. Java and Bedrock share the same world, economy, shops and progression.

Which Java version should I use?

The server runs Paper 1.21.11. ViaVersion may allow nearby versions as well.

Why does Bedrock need a port?

Bedrock servers use a separate UDP route. Enter port 15014 when adding PolarSMP.

Can console players join?

Yes, but some consoles need a third-party method to add external Bedrock servers.
